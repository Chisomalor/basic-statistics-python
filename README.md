# 📈 Agricultural Statistics with Python

## About This Project
Statistical analysis of Nigerian crop data including mean, median, standard deviation, correlation and groupby analysis with scatter plot visualization.

## Tools Used
- Python | Pandas | Matplotlib | Google Colab

## Key Finding
Rainfall and crop yield showed a 0.96 correlation — meaning more rainfall strongly predicts higher yield!
